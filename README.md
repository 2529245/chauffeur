# Prime Chauffeur Service Website

## Project Title
Prime Chauffeur Service Website

## Project Description
This project is a website for a chauffeur service company.

The website allows users to view company information, explore services and fleet options, check pricing, and contact the company via WhatsApp.

## Features
- Home page with company introduction
- About page with company information
- Services page showing chauffeur services
- Fleet page showing available cars
- Service rates page showing prices
- Contact page with company contact information
- WhatsApp booking button
- Responsive design for mobile and desktop

## Pages Included
- index.html
- about.html
- services.html
- fleets.html
- service-rates.html
- contact.html
- car detail pages

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP (optional backend)
- jQuery

## Plugins Used
- Superfish (menu)
- DL Menu (mobile menu)
- Font Awesome (icons)
- MasterSlider (homepage slider)
- Slick Slider (car carousel)

## How to Run the Project


1. Download and unzip the project folder
2. Install XAMPP
3. Move the unzipped project folder to:
   C:\xampp\htdocs\
4. Start Apache from XAMPP Control Panel
5. Open browser and go to:
   http://localhost/chauffeur-main/chauffeur-main/
